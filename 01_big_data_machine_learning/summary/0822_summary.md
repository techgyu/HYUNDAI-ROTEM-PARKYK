# 08/22 수업 내용 정리

## 1. 선형회귀 독립 변수와 종속 변수
- 선형회귀는 한 변수(독립 변수, 입력값)가 다른 변수(종속 변수, 예측값)에 어떤 영향을 미치는지 분석하는 방법입니다.
- 독립 변수($x$)는 예측에 사용되는 입력값이고, 종속 변수($y$)는 예측하고자 하는 결과값입니다.
- 참고: [선형회귀 분석의 기본 이해](https://cafe.daum.net/flowlife/SBU0/29)

## 2. 공분산을 표준화하는 목적
- 공분산은 두 변수의 변화 방향을 나타내지만, 단위의 영향을 받아 해석이 어렵습니다.
- 그래서 공분산을 표준화하여 **상관계수**로 변환합니다.
- 상관계수는 -1에서 1 사이의 값을 가지며, 두 변수의 선형적 관계의 강도와 방향을 직관적으로 파악할 수 있습니다.
  - 1에 가까울수록 강한 양의 상관관계
  - -1에 가까울수록 강한 음의 상관관계
  - 0에 가까울수록 관계가 약함

## 3. 실제 데이터 상관계수
- 실제 데이터 분석에서 상관계수가 0.3만 나와도 두 변수 사이에 꽤 의미 있는 관계가 있다고 볼 수 있습니다.
- 현실 데이터는 잡음이 많기 때문에, 0.3 이상의 상관계수도 충분히 주목할 만합니다.

## 4. 인과관계
- 상관관계가 있다고 해서 반드시 인과관계가 있는 것은 아닙니다.
- 인과관계의 유무는 데이터 분석가가 여러 분석 방법과 도메인 지식을 바탕으로 판단해야 합니다.
- **선형회귀 분석**을 통해 인과관계를 추정할 수 있습니다.
- 통계적으로 p-value가 0.05보다 작으면, 인과성이 있다고 판단합니다.
  - p-value는 귀무가설(관계 없음)이 참일 확률을 의미하며, 0.05 미만이면 통계적으로 유의미한 결과로 간주합니다.

## 5. CPU보다 GPU를 갖고 처리하는 것이 더 빠른 이유

CPU(중앙처리장치)는 소수의 코어(연산 유닛)로 복잡한 작업을 빠르게 처리하도록 설계되어 있습니다.  
반면, GPU(그래픽처리장치)는 수백~수천 개의 코어를 가지고 있어,  
동시에 많은 연산을 병렬로 처리할 수 있습니다.

특히 딥러닝, 머신러닝, 행렬 연산 등 대량의 데이터를 한 번에 계산해야 하는 작업에서는  
병렬 처리가 매우 중요합니다.  
GPU는 여러 데이터를 동시에 처리하므로,  
CPU보다 훨씬 빠른 속도로 대규모 연산을 수행할 수 있습니다.

예를 들어, 이미지 처리, 신경망 학습, 대규모 행렬 곱셈 등에서는  
CPU가 하나씩 순차적으로 계산하는 반면,  
GPU는 여러 연산을 동시에 처리해 시간 단축 효과가 큽니다.

> 요약:  
> - CPU: 복잡한 작업에 강하지만 병렬 처리 능력은 제한적  
> - GPU: 단순 연산을 대량으로 병렬 처리, 대규모 데이터 연산에 매우 빠름  
> - 딥러닝, 머신러닝, 과학 계산 등에서는 GPU가 압도적으로
